银河麒麟操作系统设置java与go系统环境变量

环境变量设置位置

打开环境变量文件：
    sudo vim /etc/profile
或者（系统环境变量）
    sudo vim /etc/enviornment

设置java环境变量

export JAVA_HOME=/usr/java/jdk1.8.0_281
export PATH=$PATH:$JAVA_HOME/bin
export CLASSPATH=.:$JAVA_HOME/lib

设置go环境变量

export PATH=$PATH:/usr/local/go/bin

环境变量生效
